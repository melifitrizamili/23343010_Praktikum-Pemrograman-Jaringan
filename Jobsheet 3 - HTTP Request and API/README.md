## Deskripsi JobSheet

JobSheet 3 membahas konsep **HTTP Request** dan penerapan **API (Application Programming Interface)** dalam pengembangan aplikasi berbasis **Node.js**.  
Pada JobSheet ini, mahasiswa mempelajari bagaimana aplikasi berkomunikasi dengan layanan eksternal melalui protokol HTTP dengan mekanisme *request–response*, serta bagaimana mengolah data yang diterima dalam format **JSON**.

---

## Tujuan Pembelajaran

Setelah menyelesaikan JobSheet ini, mahasiswa diharapkan mampu:

- Memahami konsep dasar **HTTP Request** dan **HTTP Response**
- Mengenal metode HTTP seperti `GET`, `POST`, `PUT`, dan `DELETE`
- Memahami konsep **API** dan cara kerjanya
- Mengonsumsi data API menggunakan Node.js
- Mengolah dan menampilkan data JSON dari API eksternal
- Menggabungkan lebih dari satu API dalam satu aplikasi

---

## Konsep yang Dipelajari

- Protokol HTTP dan model *request–response*
- Metode HTTP (`GET`, `POST`, `PUT`, `DELETE`)
- API (Application Programming Interface)
- Format data **JSON**
- Penggunaan *access key* pada API
- HTTP Request menggunakan module `postman-request`
- Integrasi beberapa API dalam satu aplikasi

---

## Bagian Latihan

Pada JobSheet ini, mahasiswa secara bertahap melakukan beberapa latihan berikut:

1. Mempelajari konsep HTTP Request beserta struktur request dan response.  
2. Mengenal API dan mekanisme pertukaran data berbasis JSON.  
3. Menggunakan **Weatherstack API** untuk memperoleh data cuaca berdasarkan koordinat *latitude* dan *longitude*.  
4. Menggunakan **Mapbox API** untuk melakukan *forward geocoding*, yaitu mengonversi nama lokasi menjadi koordinat geografis.  
5. Mengimplementasikan module `postman-request` pada Node.js untuk mengirim HTTP Request.  
6. Menampilkan hasil data API seperti suhu, kondisi cuaca, dan kemungkinan hujan dalam bentuk teks pada terminal.  
7. Menggabungkan API Mapbox dan Weatherstack untuk menampilkan informasi cuaca berdasarkan lokasi pengguna.

---

## Kesimpulan

Melalui JobSheet 3 ini, mahasiswa memperoleh pemahaman yang lebih mendalam mengenai komunikasi antara **klien dan server** menggunakan **HTTP Request** dan **API**.  
Mahasiswa juga belajar bagaimana mengelola data **JSON**, menggunakan *access key*, serta mengintegrasikan lebih dari satu API dalam aplikasi Node.js.  
Hasil akhir dari JobSheet ini adalah sebuah aplikasi sederhana yang mampu menampilkan **informasi cuaca berdasarkan lokasi pengguna** secara dinamis melalui terminal.
