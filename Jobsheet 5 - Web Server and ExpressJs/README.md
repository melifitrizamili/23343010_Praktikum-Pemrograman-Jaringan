## Deskripsi JobSheet

JobSheet 5 membahas pembuatan **Web Server menggunakan Node.js dan Express.js**
sebagai framework backend.  
Pada JobSheet ini, mahasiswa mempelajari konsep dasar web server, pengelolaan
routing, penyajian konten statis dan dinamis, serta penerapan *templating engine*
**Handlebars (hbs)** untuk membangun tampilan web yang terstruktur dan dinamis.

---

## Tujuan Pembelajaran

Setelah menyelesaikan JobSheet ini, mahasiswa diharapkan mampu:

- Memahami konsep dasar **Web Server**
- Menggunakan framework **Express.js** pada Node.js
- Membuat dan mengelola **routing** pada aplikasi web
- Menyajikan konten dalam format **HTML** dan **JSON**
- Mengelola **static assets** seperti CSS, JavaScript, dan gambar
- Menerapkan **templating engine Handlebars (hbs)**
- Membuat halaman kesalahan menggunakan **wildcard route (404)**

---

## Konsep yang Dipelajari

- Web Server statis dan dinamis
- Framework Express.js
- Routing dengan `app.get()`
- Static files dengan `express.static()`
- Template Engine Handlebars (hbs)
- Views dan Partials (header & footer)
- Wildcard route dan halaman 404

---

## Kesimpulan

Melalui JobSheet 5 ini, mahasiswa memperoleh pemahaman mengenai cara membangun
dan mengelola **Web Server** menggunakan **Node.js dan Express.js**.  
Mahasiswa juga mempelajari penerapan **templating engine Handlebars (hbs)** untuk
membuat tampilan web yang dinamis dan terstruktur, serta menangani berbagai rute
dan kesalahan akses halaman.  
Hasil akhir dari JobSheet ini adalah sebuah **aplikasi Web Server sederhana**
yang mampu menampilkan halaman HTML, data JSON, dan halaman kesalahan (404)
secara profesional.
