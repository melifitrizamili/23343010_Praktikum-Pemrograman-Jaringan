## Deskripsi JobSheet

JobSheet 9 membahas konsep **Socket Programming** untuk membangun aplikasi
komunikasi **real-time** berbasis web.  
Pada JobSheet ini, mahasiswa mempelajari penggunaan **WebSocket** dengan
library **Socket.IO** untuk memungkinkan komunikasi dua arah antara client
dan server secara langsung tanpa mekanisme request berulang seperti HTTP.

---

## Tujuan Pembelajaran

Setelah menyelesaikan JobSheet ini, mahasiswa diharapkan mampu:

- Memahami konsep dasar **Socket Programming**
- Menjelaskan perbedaan komunikasi HTTP dan komunikasi real-time berbasis socket
- Mengimplementasikan komunikasi dua arah menggunakan **Socket.IO**
- Mengelola event berbasis **event-driven communication**
- Membangun aplikasi chat sederhana berbasis real-time

---

## Konsep yang Dipelajari

- Socket Programming
- WebSocket
- Library Socket.IO
- Event-based communication (`socket.on`, `socket.emit`)
- Real-time bidirectional communication
- Room dan broadcast message
- Client–Server real-time interaction

---

## Kesimpulan

Melalui JobSheet 9 ini, mahasiswa memperoleh pemahaman mengenai penerapan
**Socket Programming** dalam membangun aplikasi real-time berbasis web.
Dengan menggunakan **Socket.IO**, komunikasi antara client dan server dapat
berlangsung secara dua arah, cepat, dan responsif tanpa perlu melakukan
request HTTP secara berulang.  
Hasil akhir dari JobSheet ini adalah sebuah **aplikasi chat real-time**
yang mampu mengelola pengguna, mengirim pesan, dan berbagi informasi
secara langsung dalam satu room komunikasi.
