## Deskripsi JobSheet

JobSheet 8 membahas penggunaan **MongoDB** sebagai **database server** pada aplikasi
berbasis **Node.js**.  
Pada JobSheet ini, mahasiswa mempelajari konsep database **NoSQL**, perbedaan antara
database relasional (SQL) dan NoSQL, serta penerapan MongoDB dalam pengelolaan data
berbasis dokumen yang fleksibel dan scalable.

---

## Tujuan Pembelajaran

Setelah menyelesaikan JobSheet ini, mahasiswa diharapkan mampu:

- Memahami konsep **database NoSQL**
- Mengetahui perbedaan database **SQL** dan **NoSQL**
- Menghubungkan aplikasi Node.js dengan **MongoDB**
- Memahami struktur **database, collection, dan document**
- Menggunakan **ObjectId** sebagai identitas data
- Melakukan operasi **CRUD** pada MongoDB
- Menggunakan MongoDB Compass sebagai alat bantu visual

## Kesimpulan

Melalui JobSheet 8 ini, mahasiswa memperoleh pemahaman dan pengalaman praktis dalam
menggunakan **MongoDB** sebagai database server pada aplikasi Node.js.  
Mahasiswa mempelajari perbedaan antara database SQL dan NoSQL, cara mengelola data
berbasis dokumen, serta melakukan operasi **CRUD** secara langsung menggunakan
MongoDB Native Driver.  
Hasil akhir dari JobSheet ini memberikan pemahaman menyeluruh mengenai peran
**database server** dalam aplikasi backend serta kemampuan mengelola data secara
efisien menggunakan MongoDB sebagai solusi NoSQL modern.
